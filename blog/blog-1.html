The Difference Between Margin, Border and Padding

- Posted 30 June 2022, Brayden Drummond

The CSS Box Model

The CSS box model helps us understand the differences between padding, border and margin. The term "box model" is used when talking about design and layout in CSS. The CSS box model is essentially a box that wraps around every HTML element and consists of margins, borders, padding, and the actual content. So what is Content?

The content box contains the "real" content of the element, such as text, an image, or a video. Everything created in HTML is essentially a box. The dimensions are defined by the content width and the content height and often has a background colour or background image. When writing text in HTML, this colour will automatically be white.

If the box-sizing property is set to content-box (default) and the element is a block element, the content area's size can be explicity defined with the width, min-width, max-width, height, min-height, and max-height properties.

What about Padding?

The padding area is the space between the content and the edge of the box (to border). It clears an area around the content and is affected by the background colour of the box. Its dimensions are defined by padding-box width and the padding-box height.

The thickness of the padding is determined by the padding-top, padding-right, padding-bottom and padding-left properties.

What is Border?

The border area extends the padding area to include the element's borders, it is essentially the outside edge of the box. It is a border that goes around the padding and content and is affected by the background colour of the box. Its dimensions are the border-box width and the border-box height.

The thickness ofd the borders is determined by the border-width property. If the box-sizing property is set to border-box, then the border area's size can be defined with the width, min-width, max-width, height, min-height, and max-height properties. When there is a background colour or image set on a box, it extends to the outer edge of the border.

Lastly, what is Margin?

The margin area extends the border area to include an empty area used to separate an element from its neighbours (other content or boxes). It is the empty area around the border and is completely transparent, it has no background cgit pusolour by default. Its dimensions are the margin-box width and the margin-box height.

The size of the margin area is determined by the margin-top, margin-right, margin-bottom and margin-left properties.
